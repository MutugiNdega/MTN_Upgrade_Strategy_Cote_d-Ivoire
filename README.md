# MTN_Upgrade_Strategy_Cote_d-Ivoire
## 1. Defining the Question
Understand mobile product usage in select cities in the Cote d'Ivoire to inform strategy to be used by MTN in upgrading its technology infastructure
## 2. Understanding the context
As Data Scientist for MTN Cote d'Ivoire, a leading telecom company and I am tasked with solving for the following research question:
Currently MTN Cote d'Ivoire would like to upgrade its technology infrastructure for its mobile users in Ivory Coast.
Studying the given dataset, how does MTN Cote d'Ivoire go about the upgrade of its infrastructure strategy within the given cities?
## 3. Recording the experimental design
1. Load datasets
2. Check for missing data
3. Check for duplicates
4. Drop duplicates if present
5. Combine telcom datasets
6. Merge telcom dataset with geo data on site code attribute
7. Explore the most used cities in the three days
8. Explore which cities were most used during business and working hours
9. Determine which is the most used product
